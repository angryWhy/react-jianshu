###### js 

styled-components,使用a元素

```javascript
styled.a.attrs({href:"/"})
//参数为对象，href指向根路径
```

styled-components

```javascript
export const NavItem=styled.div
&.left{
    float:left;
}
//&.意思为：当先项并且类名为left，样式生效
```

### 1.12

##### 问题：validateDOMNesting(...): 

```javascript
validateDOMNesting(...): <a> cannot appear as a descendant of <a>
```

问题原因：<Link>里面又包含了<Link>，不能

```
<a><a>22</a></a></a>
//这样嵌套
```

查找：

```javascript
<Link to="/">
<Logo/>
</Link>
改为
<Logo href="/"/>
```
